CIS Controls v8 — Safeguard Mapping

The CIS Critical Security Controls are a prioritised set of 18 top-level controls containing 153 safeguards. They are organised into three Implementation Groups (IG1, IG2, IG3) to enable scalable adoption. Below is the mapping of Microsoft Defender XDR capabilities to relevant CIS safeguards.

CIS 1 — Inventory and Control of Enterprise Assets

IG1 ▼
1.1 Establish and maintain detailed enterprise asset inventory Compliant MDE Device Inventory
1.2 Address unauthorised assets Compliant MDE Device Discovery

CIS 2 — Inventory and Control of Software Assets

IG1 ▼
2.1 Establish and maintain a software inventory Compliant MDVM Software Inventory, MDCA Shadow IT
2.3 Address unauthorised software Partial MDCA App Governance, MDVM Browser Extensions

CIS 3 — Data Protection

IG1 ▼
3.1 Establish and maintain a data management process Non-Compliant DLP Integration (not enabled)

CIS 4 — Secure Configuration

IG1 ▼
4.1 Establish and maintain a secure configuration process Partial MDI DC Sensor, MDVM Security Baselines

CIS 5 — Account Management

IG1 ▼
5.2 Use unique passwords Compliant MDI Credential Detection
5.4 Restrict administrator privileges Partial MDI AD Security Posture

CIS 6 — Access Control Management

IG1 ▼
6.8 Define and maintain role-based access control Compliant XDR RBAC, MDCA Session Controls

CIS 7 — Continuous Vulnerability Management

IG2 ▼
7.1 Establish and maintain a vulnerability management process Compliant MDVM Vulnerability Assessment

CIS 8 — Audit Log Management

IG1 ▼
8.2 Collect audit logs Compliant XDR Advanced Hunting (KQL)
8.11 Conduct audit log reviews Compliant MDI Identity Threat Detection

CIS 9 — Email and Web Browser Protections

IG1 ▼
9.2 Use DNS filtering services Partial MDE Network Protection
9.3 Maintain and enforce network-based URL filters Partial MDE Web Content Filtering, MDO Safe Links
9.5 Implement DMARC Non-Compliant MDO Email Auth (DMARC/DKIM/SPF)
9.6 Block unnecessary file types Compliant MDO Safe Attachments

CIS 10 — Malware Defences

IG1 ▼
10.1 Deploy and maintain anti-malware software Compliant MDE Endpoint Detection & Response
10.4 Configure automatic anti-malware scanning of removable media Compliant MDE Controlled Folder Access
10.5 Enable anti-exploitation features Compliant MDE Attack Surface Reduction

CIS 13 — Network Monitoring and Defence

IG2 ▼
13.5 Manage access control for remote assets Compliant MDI Lateral Movement Detection

CIS 14 — Security Awareness and Skills Training

IG1 ▼
14.1 Establish and maintain a security awareness program Partial MDO Anti-Phishing Policies

CIS 16 — Application Software Security

IG2 ▼
16.10 Apply secure design principles in application architectures Compliant MDCA OAuth App Monitoring

CIS 17 — Incident Response Management

IG1 ▼
17.4 Establish and maintain an incident response process Compliant MDE AIR, XDR Incident Queue
17.5 Assign key roles and responsibilities Compliant MDO Zero-hour Auto Purge
17.8 Conduct post-incident reviews Compliant XDR Auto Investigation & Response
17.9 Establish and maintain security incident thresholds Compliant XDR Threat Analytics Reports

CIS 18 — Penetration Testing

IG3 ▼
18.1 Establish and maintain a penetration testing program Partial XDR Secure Score
18.3 Remediate penetration test findings Partial XDR Multi-Tenant Management
